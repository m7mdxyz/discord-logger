About This Dashboard

This dashboard provides a comprehensive view of your Discord server's activity logs. It allows you to monitor message deletions, edits, voice channel activity, and member events in an organized and user-friendly interface.

Currently, the dashboard reads from JSON log files. In future versions, integration with a SQL database will be implemented for improved performance and querying capabilities.
